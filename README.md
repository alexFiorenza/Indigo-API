# Indigo API

## Features

- Database: MongoDB
- Mongoose: MongoDB ODM
- Backend Language: NodeJS (javascript)
- [Mercadopago api](https://github.com/mercadopago/dx-nodejs)
- Image Uploading
- Heroku hosting
